## 面板功能

### 核心模块
- **当前状态**：Agent 状态机实时状态
- **进化指标**：置信度标注率、幻觉率、技能熟练度
- **智能触发器**：5 种触发器统计数据
- **项目进度**：所有项目完成度
- **今日任务**：任务列表和完成状态
- **时间线**：今日活动记录

### 数据更新
- 数据文件：`dashboard-data.json`
- 更新频率：每日进化任务（21:00）自动推送
- 面板刷新：每分钟自动读取最新数据

## 技术栈

- **前端**：纯 HTML + CSS + JavaScript
- **数据**：JSON 数据文件
- **托管**：Gitee Pages
